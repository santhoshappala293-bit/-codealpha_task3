# CodeAlpha IoT Internship - Task 3

## Project Title
Smart Temperature Monitoring System Using Arduino and DHT11

## Objective
To monitor temperature and humidity in real time using a DHT11 sensor and Arduino.

## Components Used
- Arduino Uno
- DHT11 Sensor
- Breadboard
- Jumper Wires

## Working
The DHT11 sensor measures temperature and humidity. Arduino reads the sensor values and displays them on the Serial Monitor every 2 seconds.
